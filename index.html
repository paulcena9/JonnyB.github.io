General Introduction

I’m Jonny Bachman, currently an MRI Support Specialist at UCSF’s Neuroscape Lab. With a background in physics and computer science, I help maintain and optimize our Siemens Prisma 3T scanner workflow—troubleshooting pulse sequences, refining trigger latencies, and ensuring reliable data acquisition for our research teams.

My experience spans:

  • MRI Hardware & Software Integration: Designed microcontroller‐based trigger interfaces, implemented TTL pulse conversion, and configured fiber‐optic synchronization systems for fMRI experiments.
  • Neuroscience & Machine Learning: Contributed to computer vision–based Parkinsonian symptom quantification projects; developed Python pipelines for Arctic permafrost analysis and random forest models for geospatial classification.
  • Full‐Stack Development: Built Swift-based iPad apps for non‐verbal communication, crafted web scraping tools in Python to analyze used car listings, and prototyped AI dispatch tools for wildland firefighting.
  • OpenAI & AI Integration: Integrated large language models into collaborative visualization platforms; designed predictive‐text interfaces for RPM spelling communication.

Outside the lab, I’m passionate about neurophilosophy, exploring questions of consciousness through AI and computational models, and I’m always on the lookout for new ways to blend technology with human well‐being.

Projects

Project One: MRI Trigger Interface

Developed an Arduino-based TTL pulse conversion system to synchronize behavioral tasks with Siemens Prisma triggers. Refined timing accuracy to under 5 ms for fMRI event‐related designs.

Project Two: Permafrost ML Analysis

Created a Python pipeline using random forest classification and drone-based RGB imagery to monitor Arctic tundra polygon changes. Achieved 85 % accuracy in distinguishing high‐centered vs. low‐centered polygons.

Project Three: RPM Predictive Text

Built a SwiftUI-based letterboard that integrates OpenAI’s API for predictive‐text suggestions. Designed for rapid spelling communication (RPM) with customized prefix‐matching and context awareness.
